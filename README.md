## 📌 Descripción

Este proyecto es un **sistema de gestión para molino**, diseñado para facilitar el control de:

- 🌽 Procesamiento de maíz  
- ⚖️ Control de peso  
- 🧾 Ventas y servicios  
- 👥 Clientes  
- 💰 Ingresos  

Ideal para negocios tradicionales que buscan digitalizar sus operaciones.

---

## 🚀 Características

- Registro de entrada de materia prima  
- Control de molienda (kg procesados)  
- Cálculo automático de costos  
- Registro de ventas  
- Gestión de clientes  
- Historial de operaciones  
- Interfaz sencilla y rápida
